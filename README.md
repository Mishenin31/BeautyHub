# Beauty Hub — WPF-приложение «Салон красоты»

Приложение для управления салоном красоты с авторизацией и тремя ролями.

## Роли и доступ

| Раздел        | Админ | Сотрудник | Пользователь |
|---------------|:-----:|:---------:|:------------:|
| Записи        |   ✓   |     ✓     |  только свои |
| Услуги        |   ✓   |     ✓     |  просмотр    |
| Мастера       |   ✓   |     ✓     |      —       |
| Клиенты       |   ✓   |     ✓     |      —       |
| Пользователи  |   ✓   |     —     |      —       |
| Статистика    |   ✓   |     —     |      —       |

- **Администратор** — полный доступ, управление пользователями.
- **Сотрудник** — работа с записями, услугами, мастерами, клиентами.
- **Пользователь (клиент)** — личный кабинет: свои записи и онлайн-запись.

## Вход и регистрация

При запуске открывается окно входа (`LoginWindow`) с двумя вкладками:
- **Вход** — логин и пароль.
- **Регистрация** — новый клиент (автоматически роль «Пользователь» + карточка клиента).

### Тестовые учётные записи
| Логин      | Пароль     | Роль          |
|------------|------------|---------------|
| `admin`    | `admin`    | Администратор |
| `employee` | `employee` | Сотрудник     |
| `user`     | `user`     | Пользователь  |

> Пароли хранятся в открытом виде — это учебный пример. В реальном проекте используйте хеширование (например, BCrypt) и БД.

## Возможности
- **Записи** — создание, удаление, отметка о завершении
- **Услуги** — каталог с ценой, категорией и длительностью
- **Мастера / Клиенты** — справочники
- **Пользователи** — создание учётных записей с выбором роли (только админ)
- **Мои записи** — личный кабинет клиента с онлайн-записью
- **Статистика** — выручка, кол-во записей, популярность услуг

## Технологии
- .NET 8 + WPF
- Архитектура: code-behind + хранилище в памяти (singleton `DataStore`)
- Данные не сохраняются между запусками (для постоянного хранения подключите EF Core / SQLite)

## Как запустить
1. Установите [.NET 8 SDK](https://dotnet.microsoft.com/download) (нужна **Windows** — WPF только под Windows).
2. В папке проекта выполните:
   ```
   dotnet run
   ```
   Либо откройте `BeautyHub.csproj` в Visual Studio 2022 и нажмите F5.

## Структура проекта
```
BeautyHub/
├── BeautyHub.csproj
├── App.xaml / .cs            — точка входа (старт с LoginWindow), стили, палитра
├── DataStore.cs             — хранилище + аутентификация + тестовые данные
├── Models/Models.cs         — UserRole, User, Service, Master, Client, Appointment
└── Views/
    ├── LoginWindow          — вход и регистрация
    ├── MainWindow           — окно с меню по роли + выход
    ├── AppointmentsView     — записи (админ/сотрудник)
    ├── MyAppointmentsView   — личный кабинет клиента
    ├── ServicesView         — услуги
    ├── MastersView          — мастера
    ├── ClientsView          — клиенты
    ├── UsersView            — пользователи (админ)
    └── StatsView            — статистика
```

## Идеи для доработки
- Хеширование паролей и хранение в SQLite (EF Core)
- Проверка занятости мастера на выбранное время
- Редактирование записей и профиля
- Запрет доступа к чужим записям на уровне данных
